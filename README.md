# django_vue_docker_local
   - コンテナ
   `docker-compose exec vue_web sh`
   - 脆弱性の修正
   `npm audit fix`
   - Vuexのインストール
   `npm install vuex`
   - VueRouterのインストール
   `npm install vue-router`
   - Sassのインストール
   `npm install sass-loader node-sass`
   - axiosのインストール
   `npm install axios`
 - 実行に必要な環境
   - vue: 2.6.11
   - vuex: 3.5.1 
   - vue-router: 3.4.3
